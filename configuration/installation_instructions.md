The EveryAPI context server needs an API key.

1. Sign in at [app.everyapi.ai](https://app.everyapi.ai) and open **API Keys**.
2. Create a key (it starts with `sk-everyapi-`) and paste it into `api_key` below.
3. `base_url` is only needed for self-hosted gateways — leave the default for everyapi.ai.

The server is read-only: it can check your balance, summarize recent usage, and list the models available through the gateway.
